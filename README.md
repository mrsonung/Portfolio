# Premium Developer Portfolio | Kumar Sonu Nigam

A high-performance, modern, and highly animated developer portfolio built with the MERN stack aesthetic.

## 🚀 Key Features
- **Dark Futuristic UI**: Sleek dark mode with glassmorphism and neon accents.
- **Smooth Scrolling**: Integrated with Lenis for a premium browsing feel.
- **Advanced Animations**: Powered by Framer Motion for section reveals, magnetic buttons, and page transitions.
- **Interactive Components**: 
    - Custom animated cursor
    - Infinite technology marquee
    - Floating social sidebar
    - Magnetic UI elements
- **Professional Sections**:
    - **Hero**: Dynamic typing roles and clear CTAs.
    - **About**: Professional bio focused on engineering skills.
    - **Skills**: Categorized technical expertise with animated progress bars.
    - **Projects**: Redesigned showcase for Eventra, EduBridge, and BullGains.
    - **Experience**: Clean vertical timeline for professional growth.
    - **Contact**: Integrated with **EmailJS** and **react-hot-toast** for real-time inquiries.

## 🛠️ Tech Stack
- **Frontend**: React.js, Tailwind CSS
- **Animations**: Framer Motion, Lenis Scroll
- **Icons**: React Icons, Lucide React
- **Notifications**: React Hot Toast
- **Email Service**: EmailJS

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mrsonung/Portfolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Variables**:
   Create a `.env` file in the root directory and add your EmailJS credentials:
   ```env
   REACT_APP_EMAILJS_SERVICE_ID=your_service_id
   REACT_APP_EMAILJS_TEMPLATE_ID=your_template_id
   REACT_APP_EMAILJS_PUBLIC_KEY=your_public_key
   ```

4. **Start the development server**:
   ```bash
   npm start
   ```
